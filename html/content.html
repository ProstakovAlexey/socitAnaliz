Результат

На каждом графике указано время выполнения теста. В тесте несколько запросов (4-ре для 409, больше 12 для ПГУ, и много действий для веб-интерфейса) время на графике для теста в целом. Нормальное время:

  1. 409 сервис: 3-5 сек
  2. ПГУ сервис: 5-10 сек
  3. Веб-интерфейс: 40-60 сек
Предельное время для СМЭВ запросов - 2 мин (после этого в СМЭВ считается, что сервис не отвечает). Если время меньше нормального - ТИ не нагружена, запросы попали в кэш SQL сервера. Если выше - есть нагрузка на ТИ, SQL сервер выполняет какие-то операции. Время не должно ни когда превышать предельное.
При наличии ошибок на графике появляются красные полосы, кол-во ошибок на шкале справа. Шкала может показать только 5 ошибок. Под ошибкой понимается не соответствие ответа контрольному примеру из документации сервиса, либо отсутствие ответа.
