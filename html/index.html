Данные о производительности веб-сервисов и веб-интефейса АС "Соцпортал"

Показывает результаты тестов за последние 7 дней по нескольким ТИ. Нужные можно выбрать в навигаторе. Тестирование выполняется каждый час: в 10:30, 11:30, 12:30 и т.д. результаты поступают в БД. Построение графиков выполняется по данным из БД четыре раза в час (сделано на случай ручного запуска тестов).

Навигация

  1. Главная
  2. Социнформтех (ТУ)
  3. Тула
  4. Липецк

Порядок работы

Выберите нужный регион, откроется страница с его данными. На этой странице могут быть видны несколько графиков: ПГУ, 409, веб-интерфейс, для каждой ИС кол-во графиков индивидуально и зависит от выполняемых для нее тестов.
